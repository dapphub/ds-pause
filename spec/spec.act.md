## Operations

### `plot`

```act
behaviour plot of DSPause
interface plot(address usr, bytes32 tag, bytes fax, uint eta)

types

  Can: uint256
  Delay: uint256
  Plotted: uint256

storage

  delay |-> Delay
  wards[CALLER_ID] |-> Can
  plans[#hash(usr, tag, fax, eta)] |-> Plotted => 1

iff in range uint256

  TIME + Delay

iff

  Can == 1
  VCallValue == 0
  eta >= TIME + Delay

if

  sizeWordStackAux(fax, 0) == 64
  sizeWordStackAux(CD, 0) == 64
```
